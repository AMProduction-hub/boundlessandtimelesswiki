## **Roll Success and Fails Criteria**

Pada campaign ini akan diberikan beberapa kriteria-kriteria yang akan berlaku pada hasil roll dari pemain. Berikut adalah kriterianya;

| Criteria         | Formula                     |
| ---------------- | --------------------------- |
| Critical Success | $Roll + Modifier ≥ DC + 10$ |
| Success          | $Roll + Modifier ≥ DC$      |
| Failure          | $Roll + Modifier < DC$      |
| Critical Failure | $Roll + Modifier < DC + 10$ |

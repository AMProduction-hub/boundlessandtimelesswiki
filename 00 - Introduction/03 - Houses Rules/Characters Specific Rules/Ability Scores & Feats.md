# **Ability Scores Improvement**
Pada aturan (Core Rules) Dungeon and Dragons 5e setiap 4 level character dapat memilih Ability Scores Improvement atau Feats. Pada campaign ini akan diganti untuk ASI (Ability Scores Improvement) +2 akan pasti player dapatkan pada setiap level berikut:

| Levels | Ability Scores Improvement |
| :----: | :------------------------: |
|   2    |  +2 Point (Bisa disebar)   |
|   6    |          +2 Point          |
|   10   |          +2 Point          |
|   14   |          +2 Point          |
|   18   |          +2 Point          |
# **Feats** 
Untuk Feats, player mendapatkan feats setiap 4 level yang di dapat (Bukan level per Class). Berikut adalah tabel yang player character dapatkan:

| Levels |  Feats  |
| :----: | :-----: |
|   4    | 1 Feats |
|   8    | 1 Feats |
|   12   | 1 Feats |
|   16   | 1 Feats |

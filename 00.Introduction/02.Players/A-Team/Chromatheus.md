# [[Chromatheus]]
## Overview
### Basic Background 
- Nama: Chromatheus 
- Ras: Genasi Api 
- Kelas: Warlock (Fiend Patron) 
- Sifat: Lawful Neutral 
- Hometown: Shandora 
- Clothing: Pakaian berbulu putih. Ia suka memakai mantel tebal dengan bulu serigala putih. 
- Personal History Before Adventure: Sesosok genasi yang dipuja puja oleh genasi lainnya karena Chromatheus merupakan genasi api biru dan berbeda dari genasi api lainnya yang berwarna merah. Ia dipuja layaknya dewa, para genasi lain menganggapnya sebagai titisan langsung dari dewa. 
- Penyesalan: Ia tidak pernah menyesali kelahirannya, serta bagaimana warga Shandora memujanya. Namun kian lama, rasa kesombongan mulai menggerogoti hatinya, hingga ia memutuskan untuk pergi secara diam diam dari Shandora.
- Character Motivation Tujuan: Chromatheus hanya ingin membantu siapapun yang ia temui selama perjalanan, tanpa pemujaan berlebih. 
- Ketakutan: Para rakyat Shandora, adalah kelemahan hatinya.

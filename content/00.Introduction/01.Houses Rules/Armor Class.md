# **[[Armor Class]]**
Untuk armor class akan berfungsi sepertinya pada core rules dari DnD 5e. 
Berikut adalah tambahan tipe Armor Class:

## **Touch Armor Class (AC)**
Touch Armor Class (AC) adalah Armor Class tanpa menambahkan modifier dari armor yang digunakan oleh character tetapi menambahkan modifier dari *Dexterity* ataupun jika menggunakan *Unarmored Defense* dari monk maka akan menambahkan modifier dari *Wisdom*.
### **Cara menghitung Touch AC**
**Touch AC = 10 + Dexterity Modifier + Any Dodge/Deflection Bonuses + Size Modifier**

### **Cover Mechanic**
Player dapat menggunakan cover yang tersedia pada map (battle) yang akan menambahkan Touch AC character yang dapat dilihat dari:
$Cover Size$ vs $Character Size$. 
Berikut adalah attributenya:

| Cover Size                  | Touch AC Bonus       |
| :-------------------------- | -------------------- |
| Half-Cover (Setengah Badan) | +2 Touch AC          |
| 3/4 Cover                   | +5 Touch AC          |
| Total Cover                 | Tidak bisa di-target |
